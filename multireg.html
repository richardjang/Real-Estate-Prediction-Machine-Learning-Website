Multiple Regression Algorithm


//Let M and B be arrays of corresponding slope and y-intercept
Len = M.Length
A[Len+1][Len] is a Matrix of Sets of Equations
R[Len] is an Array of coefficients used for Ranking

for i=1 to Len
for j=1 to Len+1
if j==Len+1
A[i][j]=0
for n=1 to Len
A[i][j] += A[i][n]
else A[i][j] = R[j]*M[j]*i + B[j]

//Let P be an array of constants
P[] = RREF(A)
return P[]
